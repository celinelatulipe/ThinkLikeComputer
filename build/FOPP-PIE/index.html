COMP 1000: Think Like a Computer¬∂

Welcome to COMP 1000!

This course is designed to introduce people to programming, which is a fundamental skill within Computer Science. This text is called ‚ÄòThink Like a Computer‚Äô because the goal is for you to think through and understand how a computer will run the lines of code in a program, whether it‚Äôs a program you wrote or a program someone else wrote. At the same time as learning about programming and how to think like a computer, we will also begin right away by thinking about people: the end users who interact with the programs you write. And when we think about people as end users, we must think about ethics, because when people interact with computers both good and bad things can result, and it‚Äôs our responsibility to consider these possibilities. Thus, this course combines Programming, Interactivity, and Ethics: PIE!
