9.10.3. Ethics & Data ProtectionÂ¶

We most often use lists for storing data, because we often want the flexibility of being able to edit and change information. However, sometimes there is important data (private personal data, health data, salaries, etc.) that needs to be protected from tampering. As a programmer, if you need to share data but you want to ensure the data isnât tampered with, you can send the data as a tuple. This allows a function to make use of the data, but not change it.
